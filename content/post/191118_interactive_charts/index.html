#Introduction In dit korte tech report laten we zien hoe je interactieve charts kan plaatsen in je post. We doen dit door gebruik te maken van de Shiny package in R, deze kan onlline woren gezet en dan in de blogpost worden weergegeven. Dit kan via https://www.shinyapps.io/, and dit worden naar deze website verzonden via de package, rsconnect. Een andere mogelijkheid voor het maken van interatieve charts is de package Plotly, het nadeel hiervaan is dat je er alleen static data mee kan weergeven. Shiny heeft het voordeel dat je ook nog het data kan manipuleren.

Hier is een voorbeeld te zien met de package Plotly.

library(plotly)

d <- diamonds[sample(nrow(diamonds), 1000), ]

p <- plot_ly(
  d, x = ~carat, y = ~price,
  color = ~carat, size = ~carat
)

p

Voor deze post maken we gebruik van Shiny example: Diamonds Explorer. De app is opgezet met 2 verschillende bestanden, het eerste bestand is (ui.R) wat de code bevat om de interface te defineeren. Het tweede bestand is (server.R), wat de code bevat om de juistte data te serveren naar de interface. Het scheiden van de code geeft meer overzicht.
